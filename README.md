# Научное наследие профессора Б. Н. Бабкина (1911–1996)

**Реконструкция и верификация исследования "Элементарное решение проблемы Гольдбаха" (1984)**

**Автор проекта:** Сергеев Владимир Авенирович, в прошлом ассистент и аспирант Поносова Аркадия Владимировича каф. мат. анализа ПГНИУ (школа Н. В. Азбелева), первый аспирант А. В. Поносова. Сын аспиранта и близкого друга профессора — Авенира Севастьяновича Сергеева.

**Описание:**
В данном репозитории опубликованы оригинальные страницы рукописи профессора Б. Н. Бабкина, сохраненные семьей Сергеевых. Программный код `babkin_final.py` реализует авторский метод "ядра" и "запрещенных прогрессий", подтверждая математическую логику исследования на контрольных примерах. составлен ИИ
